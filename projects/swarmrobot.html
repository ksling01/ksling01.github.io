Swarm Robot

The swarm robot project was constructed out of analog components and an Arduino Mega Development Board. The Mega development board was used as a finite state machine to run the autonomous robot. Once powered up the board was programmed to start moving the forward direction and look for red or blue markings on the floor via an IR light sensor. If the robot were to hit an object it would stop and move a different direction to avoid getting stuck. Once a color was detected the machine would stop and send out an IR broadcast to any other swarm robots around that it has detected a color. An LED with that color would light up and the first letter corresponding to that color would show on an 8 bit display. It would then follow that color in a line for as far as it goes. If the end was reached it would then turn around and follow the line back to the other end point. Once it has reached the other end it would send out another broadcast that it has reached the end and turn off its color sensors. the Arduino board then moves into a listening state where it will recieve any IR broadcasted by other bots. The bot is fully autonomous and provides all its own power on board.
